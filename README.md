# Pong

# Primeiro jogo meu, feito com Python e Pygame.

# funcionalidades:

- Movimento simples
- colisão com as paredes
- colisão entre formas (retângulos)
- IA simples que segue o movimento da bolinha (com margem de erro)
- Reinício automático após atualiazação do placar (com tempo de delay)
- Menu, tela de derrota e tela de vitória
- cores para os jogadores e seus respectivos "Gols"
